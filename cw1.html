Inline CSS იწერება პირდაპირ HTML თეგში style ატრიბუტის გამოყენებით მოქმედებს მხოლოდ კონკრეტულ ელემენტზე ნაკლებად რეკომენდებულია დიდი პროექტებისთვის Internal CSS იწერება style თეგში, HTML დოკუმენტის head ნაწილში მოქმედებს მთელ გვერდზე კარგია ერთი გვერდისთვის External CSS ცალკე .css ფაილში იწერება უკავშირდება HTML-ს link თეგით გამოიყენება მრავალ გვერდზე და არის საუკეთესო პრაქტიკა
